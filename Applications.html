Linux commands

The most common commands used:

  • pwd --> current path
  • ls --> list of file in the current path
  • touch --> to create a blank new file
  • nano --> edit file
  • clear --> clean the space in the command prompt/terminal
  • cd --> change the path
  • cp --> copy file
  • mv --> move file
  • midkr --> to make a new directory
  • rmdir --> delete a directory, only allows you to delete empty directories
  • rm --> to delete directories and the contents within them
  • gerp --> search through all the text in a given file
  • sudo --> to perform tasks that require administrative or root permissions
  • chmod --> used to change the read, write, and execute permissions of files and directories
  • ping --> check the connectivity status to a server
  • history --> to review the commands youâ€™ve entered before
  • echo --> move some data into a file
